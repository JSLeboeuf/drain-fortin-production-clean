## 🚀 Guide de Déploiement

### Prérequis
- Node.js 18+ installé
- Compte Supabase Pro actif
- Clés API configurées
- Domaine et SSL configurés

### Étapes de Déploiement

#### 1. Configuration Environnement
```bash
# Copier le fichier d'environnement
cp .env.example .env.production

# Configurer les variables requises
VITE_SUPABASE_URL=votre_url_supabase
VITE_SUPABASE_ANON_KEY=votre_cle_anon
VITE_VAPI_PUBLIC_KEY=votre_cle_vapi
VITE_VAPI_PRIVATE_KEY=votre_cle_privee
```

#### 2. Installation et Build
```bash
# Installation des dépendances
npm install

# Build de production
npm run build

# Le dossier 'dist' contient l'application prête
```

#### 3. Déploiement Supabase
```bash
# Lier au projet Supabase
npx supabase link --project-ref votre_projet_ref

# Déployer les Edge Functions
npx supabase functions deploy

# Appliquer les migrations
npx supabase db push
```

#### 4. Vérification Post-Déploiement
```bash
# Tests de santé
npm run test:e2e

# Monitoring
npm run monitor
```
